2. 화면별 상세 디자인 (Wireframes & Descriptions)
2.1 메인 페이지 (index.html) - 업로드 및 정보 입력
사용자가 가장 먼저 접하는 화면으로, 행동 유도(사진 업로드)가 명확해야 합니다.

[화면 구조도 - Wireframe]

|   P: AI가 칼로리를 계산하고 운동을 추천해 드립니다.   |

|   +-----------------------------------------------+   |
|   |         [아이콘: 카메라/구름]                  |   |

|   |         (JPG, PNG / Max 10MB)                 |   |
|   | --------------------------------------------- |   |
|   |  [파일 선택 버튼] (선택된 파일 없음)           |   |

2.3 결과 페이지 (result.html) - 분석 결과 및 수량 수정
업로드한 이미지, 인식된 음식 목록(수정 가능), 계산된 운동 플랜을 카드 형태로 보여줍니다.
